Tag Use
< h1-6 > Headings of various sizes
< p > A paragraph, a block of text
< img > An image, doesn't have a closing tag, requires a source, src, either a link or an image in the VSC file
href Not a tag, an attribute of an a tag, used to add a link to an < a > tag
< div > Placed around a collection of tags in order to style all of them at once
< span > Placed around a tag in order to style it
< table > Makes a table, requires the tr tag to add a row to it, requires the th tag to add bold, centered text to it, requires the td tag to add some text to it
< ol/ul > Makes a list, ol is ordered/numbered, ul is unordered/dot points, requires li to add items to it
< s-tyle > Same level as the head and body tags, used to style tags, just type a tag or #id followed by a { to style
< header > The header of a website, the section at the very top, usually a title
< nav > A navigation bar, usually horizontal, just underneath the header, usually contains links
< footer > The footer of a website, the section at the very bottom
